TikTok Collection Archiver
=========================

A tool for creating local backups of TikTok collections and favorites.

PURPOSE:
This tool helps users preserve their TikTok collections by downloading videos
and saving associated metadata locally. It's designed for content creators,
researchers, and regular users who want to maintain offline copies of their
collections.

REQUIREMENTS:
- Python 3.7 or higher
- Google Chrome browser
- Windows, macOS, or Linux
- Active TikTok account logged in via Chrome

INSTALLATION:
1. Download the repository
2. Open terminal/command prompt in the project directory
3. Run: python requirements.py

USAGE:
1. Ensure you're logged into TikTok in Chrome
2. Open your TikTok collection
3. Copy the URL of the first video
4. Run: python tt-collection-backup.py
5. Paste the URL when prompted
6. Wait for backup to complete

The tool will create a new directory with your backed-up content, including
videos, metadata, and collection information.

USE CASES:
- Backing up inspiration collections
- Archiving themed content
- Preserving favorites offline
- Creating backups before collection cleanup

Note: This tool is for personal backup purposes only. Please respect
copyright and TikTok's terms of service.
